Trust & security

GranEverest runs a single ETH vault on Base. No liquidations, no variable interest. Here is how the contract and governance are set up today.

Vault contract

The ETH vault lives on Base mainnet. It custody ETH only and applies a fixed 0.25% fee on deposit & withdrawal.
