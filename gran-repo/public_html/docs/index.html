GranEverest ETH vault · Docs

High-level documentation for the 0% interest ETH vault on Base.

Protocol overview

  • Single ETH collateral vault deployed on Base.
  • Debt unit is ETH, fixed max LTV at 70%.
  • Protocol fee: 0.25% on deposit and withdrawal only.
  • No interest on debt. Borrow/repay have no protocol fee.
  • No liquidation engine and no price oracle.

Core flows

  1. Deposit. User sends ETH to the vault. Their collateral and borrow limit are updated. Fee 0.25% to treasury.
  2. Borrow. User borrows ETH up to 70% LTV. No interest, no protocol fee.
  3. Repay. User sends ETH back to reduce debt (no fee). This increases available borrow and withdrawable collateral.
  4. Withdraw. User withdraws ETH as long as LTV ≤ 70%. A 0.25% fee applies to the withdrawn amount.

Risk model

  • No liquidations: the protocol never force-sells collateral.
  • Users must manage their own LTV; if ETH price moves, their withdrawable amount changes.
  • Anti-loop guard blocks same-block deposit+borrow loops to avoid misleading "infinite credit" behaviour.

Implementation notes

  • Contracts deployed on Base mainnet.
  • Owner/guardian controlled by a hardware-wallet EOA.
  • Source code and tests are published on the GitHub repository.
